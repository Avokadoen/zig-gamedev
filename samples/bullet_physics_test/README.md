## Bullet physics test

**Left Mouse Button** :  select object<br />
**Left Mouse Button + drag** :  pick up and move object<br />
**Right Mouse Button + drag** :  rotate camera<br />
**W, A, S, D** :  move camera<br />
**SPACE** :  shoot<br />
